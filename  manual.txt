
СТРУКТУРА:
tg_bot
├── bot.py - главный файл запуска бота
├── handlers 
|   ├── admin.py - обработчики для администратора
|   ├── client.py - обработчики для клиентов 
|   └── common.py - общие обработчики  
├── keyboards - папка с клавиатурами
│   ├── admin_kb.py 
│   └── client_kb.py
├── middlewares 
|   └── db.py - подключение к базе данных
├── utils.py - вспомогательные функции
├── config.py - настройки бота 
└── requirements.txt - список библиотек




ВСЕ ЧЕРЕЗ ТЕРМИНАЛ:

работает именно на этой версии библиотеки:

pip3 install python-telegram-bot==13.7

список всех библиотек:

 pip3 list  

 команда запуска:

 через: VS Code

python3 bot.py


Чтобы остановить бота, вы можете нажать комбинацию клавиш 
Ctrl + C в терминале, где запущен скрипт бота. 
Это приведет к остановке выполнения скрипта и остановке работы бота.
